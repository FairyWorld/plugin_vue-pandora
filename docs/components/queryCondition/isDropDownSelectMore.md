# 以下拉方式展示更多条件

1.设置`isDropDownSelectMore`

2.`popoverAttrs`默认值：

```ts
{
    showTxt: '更多',
     title: '所有条件',
     allTxt: '全选',
     reverseTxt: '反选',
     clearTxt: '清空',
     placement: 'bottom',
     width: 240,
     trigger: 'click',
     ...this.popoverAttrs
}
```
