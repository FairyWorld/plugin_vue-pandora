# 弹窗组件

---

```markdown
在组件中需配置：

属性`visible`是否显示弹窗

`title`：弹窗标题（默认：温馨提示）

`@update:visible` 右上删除或取消事件

`isShowDialogDrag`属性可以取消头部拖拽功能

`ElementUI`中`el-dialog`的属性皆可用(如：`close-on-click-modal`属性——点击空白区域是否关闭弹窗)
```
